## AI Implementation Rules
- Prioritize clarity, trust, and professional hierarchy.
- Keep spacing consistent using a structured scale.
- Use the Primary color for main navigation, key headings, and strategic emphasis.
- Use Secondary color sparingly for highlights and calls to action.
- Use Accent color for interactive states, links, and supporting emphasis.
- Maintain strong contrast for accessibility.
- Ensure responsive behavior on desktop, tablet, and mobile.
- Use one shared header component and one shared footer component site-wide.
- Reuse a consistent CTA component style and messaging structure on major pages.
